# json_view
在线的一款好用json格式化工具

访问地址:
https://floweroda.github.io/json_view/
# 功能
- 支持jsonpath检索和路径复制
- 支持正则、大小写、全字匹配检索
- 支持删除节点
- 支持 匹配数据多种显示方式
- 支持将部分超长字段替换为...
